# Тестовое задание для компании Cross Technologies
Проект включает в себя веб-приложение и 3 вспомогательных проекта, в рамках которых было реализовано разделение приложения на слои.
## Слои приложения
### Доменный слой
В нем хранятся сущности приложения, ValueObject'ы, перечисления и т.д.
### Слой приложения
В этом слое заданы интерфейсы, с помощью которых разные слои смогут взаимодействовать друг с другом, к примеру, интерфейсы UseCase, Gateway, вспомогательных сервисов.

В этом слое также реализованы правила валидации поступающих запросов, отображение сущностей в модели, сами UseCase.
### Слой инфраструктуры
В этом слое предполагается хранить тот функционал, который не относится к проекту напрямую и легко может быть заменен идентичным.

В данном случае там реализованы Gateway, которые выполняют роль хранилища данных.
### Слой представления
В этом слое хранится функционал, с которым напрямую может работать пользователь. В нем отсутствует бизнес-логика.

В данном случае роль приложения представления играет веб-приложение.

Связь с UseCase и Gateway установлена через отражения, реализованные в методе расширения.
## Требования
* ASP.NET Core 3.0 MVC
